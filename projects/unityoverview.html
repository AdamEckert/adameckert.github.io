Overview

This was a short tutorial document I created to help introduce our new members to using the Unity interface. Using a sample project provided with the overview, it teaches how to open a project from Unity Hub, edit game elements, run the game in Unity, and work between multiple different scenes.

Using this guide, one can go all the way form not knowing a thing about Unity to learning to make your own levels in an existing project, and I'm very satisfied with the path I led users down to interact with each section of the interface. This was especially facilitated by the visual aids documenting each step. When using this guide to train new design team members, it served its purpose very effectively and saw basically no issue whatsoever.
