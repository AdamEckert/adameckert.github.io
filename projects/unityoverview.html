Overview

This was a short tutorial document I created to help introduce our new members to using the Unity interface. With the sample project provided alongside the overview, it teaches how to open a project from Unity Hub, edit game elements, run the game in Unity, and work between multiple different scenes.

Using this guide, members who were not familiar with Unity learned how to create their own levels in the provided sample project, with visual aids directing them through all the essential parts of the program they would need to use.
